JavaScript Statements

Statements are composed of:

  • Values
  • Operators
  • Expressions
  • Keywords
  • Comments

JavaScript Values

JavaScript has two types of values: fixed and variables.

Fixed values are known as literals while variable values are known as variables.

JavaScript Literals

Numbers can be written with or without decimals:

10.5 1001

Strings are text, written within double or single quotes:

"John Doe" 'John Doe'
